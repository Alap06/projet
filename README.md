# 🍽️ Diary — Tunisian Homemade Marketplace

Diary is a modern, high-performance marketplace connecting passionate home cooks with local customers in Tunisia. Experience the authentic taste of home-cooked meals, delivered straight to your door.

## 🚀 Key Features

- **Dynamic Marketplace**: Browse and filter dishes by category, region, and price.
- **Secure Ordering**: Seamless cart and checkout experience with multiple payment options (Cash on delivery, D17, Card).
- **Cook Dashboard**: Dedicated space for cooks to manage their menu, track orders, and coordinate deliveries.
- **Admin Supervision**: Powerful administration panel for dish validation and platform oversight.
- **Performance First**: Built with TanStack Start for lightning-fast SSR and edge-ready deployment.

## 🛠️ Technology Stack

- **Framework**: [TanStack Start](https://tanstack.com/start) (React 19 + TanStack Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **State Management**: React Context + TanStack Query
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) (Radix UI)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: Vite

## 📦 Getting Started

### Prerequisites

- Node.js 20+ or Bun
- npm or pnpm or bun

### Installation

1. **Clone the repository**:

   ```bash
   git clone <repository-url>
   cd projet
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Start the development server**:

   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
